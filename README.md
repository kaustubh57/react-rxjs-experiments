## Start the app:
  - In the project directory, run `npm install`.
  - Start the app with `npm start`
    - Runs the app in the development mode.<br />
    - Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Project dependencies:
  - redux
  - react-redux
  - rxjs
  - redux-observable
  - lodash
